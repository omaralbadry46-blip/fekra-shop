# متجر فكرة الإلكتروني | Fekra Shop

موقع إلكتروني بسيط وعصري (Minimalist) لبيع المنتجات المختلفة، مصمم بهوية بصرية مميزة تعتمد على اللونين الأسود والذهبي وخطوط عربية وأرقام هندسية عصرية.

---

## 🎨 الهوية البصرية ونظام التصميم
- **الألوان المستخدمة**:
  - الذهبي البراق: `#eeba2c` (كلون للتفاعل والأسعار والـ Buttons)
  - درجات الأسود والرمادي الداكن: `#000000` (الخلفية)، `#161616` (الحاويات والأقسام)، `#212121` (البطاقات والمدخلات).
- **الخطوط**:
  - خط **Cairo**: للعناوين والنصوص العربية المنسقة.
  - خط **Outfit**: للأرقام، الأسعار، والنصوص الإنجليزية لإعطاء مظهر عصري فاخر.

## 🚀 كيفية تشغيل الموقع محلياً
يمكنك تشغيل الموقع مباشرة بفتح ملف `index.html` في المتصفح!
يحتوي الموقع على **وضع تجريبي مدمج (Sandbox)** يتيح لك تجربة إضافة المنتجات وإضافتها للسلة وإتمام الطلبات محلياً دون الحاجة لإعداد قواعد البيانات مسبقاً.

للاختبار السريع للوحة التحكم بدون Firebase:
- رابط لوحة التحكم: افتح ملف `login.html` أو اضغط على "لوحة الإدارة" في الهيدر.
- الحساب التجريبي: `admin@fekra.com`
- كلمة المرور التجريبية: `admin123`

## 🛠️ الربط البرمجي بقواعد البيانات (Firebase & Supabase)

لتفعيل استقبال الطلبات ورفع صور منتجاتك بشكل حقيقي، اتبع الخطوات البسيطة التالية:

### أولاً: إعداد Firebase (لقاعدة البيانات والمصادقة)
1. اذهب إلى [Firebase Console](https://console.firebase.google.com/) وأنشئ مشروعاً جديداً باسم **Fekra Shop**.
2. من القائمة الجانبية، اذهب إلى **Authentication** -> **Sign-in method** -> وقم بتفعيل خيار **Email/Password**.
3. اذهب إلى **Users** وأضف حساب مشرف جديد (البريد الإلكتروني والرقم السري اللذين ستستخدمهما لدخول لوحة التحكم).
4. اذهب إلى **Firestore Database** واضغط على **Create Database**، ثم اختر وضع البدء (يمكنك اختيار Test Mode للبدء السريع).
5. من صفحة إعدادات المشروع (Project Settings)، قم بإضافة تطبيق ويب واحصل على كود التهيئة (Config Script).
6. افتح ملف [firebase-config.js](file:///h:/fekra%20shop/js/firebase-config.js) واستبدل قيم المتغير `firebaseConfig` بالقيم الخاصة بمشروعك.

### ثانياً: إعداد Supabase (لرفع وتخزين الصور)
1. افتح حساباً في [Supabase](https://supabase.com/) وأنشئ مشروعاً جديداً.
2. من القائمة الجانبية اذهب إلى **Storage** وأنشئ Bucket جديد باسم `product-images` واجعله **Public** (عام) لتتمكن الصور من الظهور للعملاء.
3. من إعدادات الـ Storage، تأكد من إضافة سياسة سماح (Policies) للمستخدمين لرفع الملفات (Insert/Select/Delete).
4. اذهب إلى إعدادات المشروع (Project Settings) -> **API** وانسخ رابط الـ **Project URL** ومفتاح الـ **Anon Key**.
5. افتح ملف [supabase-config.js](file:///h:/fekra%20shop/js/supabase-config.js) واستبدل قيم `supabaseUrl` و `supabaseKey` بالقيم التي نسختها.

## 📦 الرفع والنشر على GitHub
لرفع الكود على مستودع GitHub الخاص بك، افتح سطر الأوامر (Terminal) في مجلد المشروع ونفذ الأوامر التالية:

```bash
# تهيئة مستودع جيت المحلي
git init

# إضافة جميع الملفات للمستودع
git add .

# تسجيل التغييرات
git commit -m "Initial commit - Fekra Shop"

# ربط المستودع بمستودعك على GitHub
git remote add origin https://github.com/USERNAME/fekra-shop.git

# رفع الكود للفرع الرئيسي
git branch -M main
git push -u origin main
```
*(استبدل رابط المستودع بالرابط الفعلي الخاص بك).*

## 🌐 استضافة الموقع ونشره مجاناً
يمكنك استضافة الموقع مجاناً بنسبة 100% باستخدام:
1. **GitHub Pages**: من إعدادات مستودع GitHub الخاص بك (Settings -> Pages -> اختر الفرع main ومجلد root -> حفظ).
2. **Firebase Hosting**: عن طريق تثبيت Firebase CLI وتشغيل `firebase init hosting` ثم اختيار مجلد المشروع الحالي ورفع الملفات بأمر `firebase deploy`.
